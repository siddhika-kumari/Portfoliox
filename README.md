# 🌐 Personal Portfolio Website

Welcome to my personal portfolio website repository.

This project was built to showcase my skills, projects, and learning journey as an aspiring web developer. It serves as a central place where visitors can learn more about me, explore my work, and connect with me.

---

## 📌 About the Project

This portfolio website is designed and developed using HTML and CSS with a focus on clean design, responsive layouts, and user-friendly navigation.

The website includes:

* A professional introduction section
* About Me section
* Skills section
* Projects showcase
* Contact section
* Interactive typing animation
* Fixed navigation bar with smooth scrolling

---

## 🚀 Technologies Used

* HTML5
* CSS3
* Font Awesome
* Google Fonts (Poppins)
* Typed.js
